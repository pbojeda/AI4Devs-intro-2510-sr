

> **Contexto inicial:**
> Quiero montar una **Aplicación web** que tiene como finalidad **invertir el orden de una cadena de texto** simplemente con **lenguaje html y script javascript**.
>
> **Dispongo de estos ficheros:**
> - `index.html` → fichero inicial html 
> - `script.js` → fichero aún vacío en el que debe ir la lógica 
> - `reversestring-example.png` → imagen de ejemplo de la salida
>
> **Objetivo**  
> 1. Crear el código necesario en el fichero html, de tal forma que exista un campo de texto de entrada y un botón.  
> 2. Crear el código necesario en el fichero js que tenga la lógica de funcionamiento de invertir el orden de la palabra que está en el campo de texto, de modo que si introduzco Pablo la salida debe ser olbaP
> 3. Detectar que cuando se pulse el botón se muestre la string invertida.
> 4. El usuario debe tener instrucciones en pantalla con el funcionamiento correcto: introducir al menos dos caracteres para que el botón se active, saber que a partir del cuarto caracter introducido no hará falta pulsar el botón.
>
> **Mejoras que puedes implementar**  
> 1. No permitir pulsar el botón en el caso de que el campo de texto contenga menos de dos caracteres.
> 2. En el caso de que se introduzcan más de 3 caracteres, no esperar a que se pulse el botón para realizar la transformación de la palabra e ir haciendo la transformación sobre la marcha
>
